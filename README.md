# EchoVault

Un gioco 3D in prima/terza persona di puzzle ed esplorazione creato con Babylon.js, senza build steps e caricato interamente via CDN.

## Requisiti
- Nessun build step: utilizza ES Modules nativi.
- Avvio con: `npx serve .`

## Controlli
- **WASD**: Movimento
- **Mouse**: Visuale (Pointer Lock)
- **E**: Interagisci/spingi blocco
- **F**: Cambia visuale (prima/terza persona)
- **ESC**: Menu Pausa
- **R**: Riavvia Livello
